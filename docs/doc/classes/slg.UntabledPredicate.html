Class UntabledPredicate

An untabled predicate defined by a rule. In SLG terms, it evaluates via Program Clause Resolution. Operationally, it's just a wrapper around a JsonTerm to LPTerm function. The hard work is done by the operations which build LPTerms. This works exactly like normal Prolog execution with all the pitfalls that implies, e.g. infinite loops in left recursion.
