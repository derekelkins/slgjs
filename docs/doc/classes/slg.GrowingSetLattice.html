Class GrowingSetLattice

A semi-lattice with unit the empty set and join set union. It begins empty and becomes increasingly large as body produces (distinct) elements.

join

  • join(In: any, Out: any): { for: any }

  • Inherited from BaseLattice.join

    • Defined in slg.ts:1216

    This essentially calculates: Out = join(x, In) where join is the semi-lattice join operation, and x is the "greatest" value for the lattice term seen so far, where the ordering is the one induced by the semi-lattice operations.

    More specifically, we have a separate lattice term for each row, so lattice.join(In, Out).for(row) uses the lattice value for the specified row.

    Parameters

    • In: any

      A JsonTerm with no unbound variables.

    • Out: any
