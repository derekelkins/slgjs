Class EphemeralUnionFind<A>

An in-place, mutable implementation of UnionFind, i.e. the standard one going back to 1970.

bindValue and bindVariable just return this.

This isn't used for anything and probably won't be. I do potentially have a use for an ephemeral union-find structure for SCC maintenance, but that use warrants different choices than are made here, so this is mostly for reference.

bindVariable

  • bindVariable(x: number, y: number): EphemeralUnionFind<A>

  • Implementation of UnionFind.bindVariable

    • Defined in puf.ts:121

    Merge the two equivalence classes of x and y. The value, if any, of the Variable representing the equivalance class of y becomes the value of the Variable representing the resulting combined equivalence class. It is an error if the Variable representing the equivalence class of x is bound, even if the Variable representing the equivalence class of y is unbound. If neither is bound, this is just the standard "union" operation.

    Parameters

    • x: number
    • y: number

    Returns EphemeralUnionFind<A>

    An updated UnionFind structure where the equivalence classes of x and y have been merged.
