VarMap

VarMap: { vars: Variable[] }
  • Defined in json-trie.ts:873

The represents a list of Variables and a mapping of their IDs onto the interval [0, N) where N is the length of the list of variables. This corresponds to a variable renaming of a term assuming one of the term's variables are have contiguous IDs starting at 0. This also means that VarMap represents a bijection as you can view the array as a mapping going the other direction.
